# Installation

## From npm

```bash
npm install @slothslasher/arcane-builder-copy
npm start
```

After installation, your `package.json` will automatically have these scripts added:
- `npm start` — Run the game on http://localhost:3000
- `npm dev` — Serve the game
- `npm build` — Display build status message

## From GitHub

```bash
git clone https://github.com/Dexter0013/Arcane-Builder_Copy.git
cd Arcane-Builder_Copy
npm install
npm run dev
```

Then visit `http://localhost:5173`

## Requirements

- Node.js v18+
- npm v9+
