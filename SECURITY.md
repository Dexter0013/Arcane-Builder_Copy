# Security Policy

## Supported Versions

- **Latest version:** v1.1.5 and above

## Reporting a Vulnerability

If you discover a security vulnerability, please email [security@example.com](mailto:security@example.com) instead of using the issue tracker.

Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if available)

## Security Considerations

- This is a client-side WebGL game application
- No user data is collected or transmitted
- All game state is stored locally in the browser
- The application runs entirely in the browser
